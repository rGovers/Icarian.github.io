ShaderBufferType

Namespace: IcarianEngine.Rendering

Parent: ushort

Enum Values

ShaderBufferType.Null = ushort.MaxValue,

Null buffer type. Used for error checking.

ShaderBufferType.CameraBuffer = 0,

Use the inbuilt camera buffer.

ShaderBufferType.ModelBuffer = 1,

Use the inbuilt model buffer.

ShaderBufferType.UIBuffer = 2,

Use the inbuilt UI buffer.

ShaderBufferType.DirectionalLightBuffer = 3,

Use the inbuilt directional light buffer.

ShaderBufferType.PointLightBuffer = 4,

Use the inbuilt point light buffer.

ShaderBufferType.SpotLightBuffer = 5,

Use the inbuilt spot light buffer.

ShaderBufferType.Texture = 6,

Will contain a texture that only needs to be bound once. Can cause issues and artifacts if rebound while used. Recommended to use PushTexture if rebinding is required.

ShaderBufferType.PushTexture = 7,

Will contain a texture that needs to be bound on use.

ShaderBufferType.SSModelBuffer = 8,

Batch all transforms into a single buffer. Uses Shader Storage Buffer Objects. Will only do one draw call per model.

ShaderBufferType.SSBoneBuffer = 9

Buffer that contains all the bones. Uses Shader Storage Buffer Objects. Only used for skinned models.

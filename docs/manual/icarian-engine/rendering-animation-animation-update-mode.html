AnimationUpdateMode

Namespace: IcarianEngine.Rendering.Animation

Parent: ushort

Enum Values

AnimationUpdateMode.None = 0,

The animation is not updated.

AnimationUpdateMode.Update = 0b1 << 0,

The animation is updated on the main thread.

AnimationUpdateMode.FrameUpdate = 0b1 << 1,

The animation is updated on the rendering thread.

AnimationUpdateMode.PooledUpdateLow = 0b1 << 2

The animation is updated on the ThreadPool with a low priority.

AnimationUpdateMode.PooledUpdateMedium = 0b1 << 3

The animation is updated on the ThreadPool with a normal priority.

AnimationUpdateMode.PooledUpdateHigh = 0b1 << 4

The animation is updated on the ThreadPool with a high priority.

DefaultRenderPipeline

Namespace: IcarianEngine.Rendering

Parent: RenderPipeline

Interfaces: IDisposable

Description

The default render pipeline for the engine. This pipeline is used when no other pipeline is specified. Uses blinn phong shading with SSAO and emission.

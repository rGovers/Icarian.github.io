RenderCommand

Namespace: IcarianEngine.Rendering

Description

Render Command is used to dispatch commands to the renderer to be executed. It has per thread state reset at the start of each frame. This is used in Render Pipelines to dispatch commands to the renderer.

Functions

static void BindMaterial ( Material a_material )

Binds a material to the renderer.

static void PushTexture ( uint a_slot, TextureSampler a_sampler )

Updates the texture sampler for the given slot of the current material. Requires ShaderBufferType.PushTexture for correct behaviour.

static void BindRenderTexture ( IRenderTexture a_renderTexture )

Binds a render texture to the renderer.

static void Blit ( IRenderTexture a_srtTexture, IRenderTexture a_dstTexture )

Blits a source render texture to a destination render texture. Use null in a_dstTexture to blit to the screen.

static void DrawMaterial ( )

Draws a material with no model information.

static void DrawModel ( Matrix4 a_transform, Model a_model )

Draws a model with the given transform. Used for materials that use ShaderBufferType.ModelBuffer.

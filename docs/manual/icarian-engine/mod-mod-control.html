ModControl

Namespace: IcarianEngine.Mod

Description

The main controller for everything in Icarian Engine.

Properties

static IcarianAssembly CoreAssembly { get; }

This is the assembly that contains everything relating to the game. This is needed for everything to work. This is loaded automatically from the Core folder.

static List<IcarianAssembly> Assemblies { get; }

This is a list of all loaded mod assemblies. This will be loaded from the mods folder and modlist.xml in future. --- FLAGGED FOR CHANGE ---

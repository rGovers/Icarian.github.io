Mods

Everything in Icarian is a mod. If you are creating a game you just happen to be the Core mod. A mod can be made up of several things Defs, Scenes, Assets, Assemblies, and Scribe files. The latter of which will be explored later.

More in depth modding info will be explored in a seperate section as not everything directly relates to building a game.
